# MazeBuilder

Just a toy project based off of an assignment from uni. 

[Demo.](http://pakkudon.github.io/MazeBuilder)

## Algorithms used

Maze-builder algorithms:

- Depth-first search
- Randomized Kruskal's
- Randomized Prim's
- Eller's

Maze-solver algorithms:

- Depth-first search
- Breadth-first search
- A* Search
- Uniform-cost Search
